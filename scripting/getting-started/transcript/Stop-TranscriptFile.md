# Stop-TranscriptFile

## SYNOPSIS
Detiene la transcripción que se inició mediante Start-TranscriptFile.

## SYNTAX

```powershell
Stop-TranscriptFile
```

## DESCRIPTION
Detiene la transcripción que se inició mediante Start-TranscriptFile.
También puede detener la transcripción finalizando la sesión.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```powershell
Stop-TranscriptFile
```
